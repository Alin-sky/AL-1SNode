    GPT AI对话

    Chat GPT相关
    首先很遗憾的告诉大家😿可能是因为AI对话插件设计的问题，
    在聊天过于频繁的时候会触发429报错，很影响大家使用体验。
    所以，从2023年8月15日起不再接受新群聊开启AI对话的申请😭😭😭
    已经开启AI对话的大群聊也会挑选并关闭
    这也是为了避免键政和风控

     GPT对话说明

     chat GPT插件给AI写入了一个爱丽丝的人设。
     爱丽丝是《Blue Archive》游戏里的角色“爱丽丝”。
     因此给chat GPT写了一个模仿爱丽丝的人设。
     私聊发送任何非指令内容都会触发这个插件
     群聊发送任何带有“爱丽丝”的文字都会触发这个插件。
     如果嫌烦群聊中可以使用同音字。
     如爱丽斯、艾丽斯、爱莉丝。als等名字来规避触发。
     chat GTP具有强大的对话功能。但回答的内容不可一味相信。
     “chat GPT可以保证有答案。但不保证答案正确”。
     chat GPT使用了大量维基百科的知识来训练。内容都没有也不可能经过国内审核。
     所以千万不要问机器人任何与zg政治、赚钱之类的问题。
     AI极有可能瞎说些敏感问题而导致QQ账号封禁。
     open AI的chat GPT服务拥有的使用额度有限。
     在使用中有可能会出现403错误。一般情况下是没有额度了。
     阿林会想办法来补充使用额度的
     chat GPT在多次对话后会爆缓存。会出现无法对话的情况。并报错429或者400
     这时候需要发送“重置”清空对话记录和缓存
